# CardPayTest

Тестовое задание - API-тесты для сервиса http://fish-text.ru/api

Для тестов использованы:
- Java 8
- Cucumber
- Rest Assured
- Maven
- JUnit

Матрица тестовых случаев приведена в гугл-доке - https://docs.google.com/spreadsheets/d/1MfyTNwozuLf9bkGdKY3H_4Cz-IFMnhN2ow511xLsJzw/edit?usp=sharing
Примечание: особенность сервиса такова, что он одинаково воспринимает как отсутствие какого-либо параметра (например, запрос без указания type), так и некорректное значение этого же параметра (спецсимволы, string вместо int для параметра number, для других параметров - значения не из разрешенного диапазона). Поэтому в тестах такие проверки были объединены в одну, дефолтную.
